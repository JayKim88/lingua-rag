# LinguaRAG — 제품 소개

> 내 교재로 배우는 AI 언어 튜터

---

## 한 줄 요약

PDF 교재를 올리면, AI가 그 교재 내용을 기반으로 질문에 답해주는 개인 언어 튜터.

---

## 누구를 위한 제품인가?

| 페르소나 | 상황 | LinguaRAG가 주는 가치 |
|----------|------|----------------------|
| 독학 언어 학습자 | 교재로 혼자 공부하는데 모르는 게 나오면 물어볼 곳이 없다 | 교재 맥락을 아는 AI가 즉시 답변 |
| 어학원 수강생 | 수업 교재 복습할 때 선생님이 없다 | 교재 기반으로 추가 설명, 예문, 연습 제공 |
| 다국어 학습자 | 독일어, 영어, 프랑스어 등 여러 언어를 배운다 | 9개 언어 지원, 교재별 독립 학습 공간 |

## 핵심 기능

### 1. PDF 업로드 → 즉시 학습 시작

- 내가 쓰는 교재 PDF를 업로드
- AI가 교재 내용을 자동으로 분석(임베딩 + 벡터 검색)
- 교재 속 단어, 문법, 예문을 기반으로 정확한 답변

### 2. 교재 맥락을 아는 AI 채팅

- **"이 페이지에서 모르는 문법 설명해줘"** → 현재 보고 있는 페이지 텍스트를 자동 인식
- 교재에 나온 예문과 단어를 활용한 맥락적 답변
- 대화 기록이 유지되어 이전 질문 맥락도 기억

### 3. PDF 뷰어 + 인터랙티브 학습

| 기능 | 동작 |
|------|------|
| 텍스트 선택 | 드래그하면 팝업 메뉴 등장 |
| 발음 듣기 | 선택한 텍스트를 원어민 발음으로 재생 |
| 번역 | 선택한 텍스트 즉시 번역 |
| 질문하기 | 선택한 텍스트를 채팅에 자동 입력 |
| 발음 연습 | 마이크로 따라 읽고 정확도 측정 |
| 메모 | 페이지에 컬러 스티키 노트 부착 |

### 4. 발음 연습 모드

- 텍스트 선택 → "연습" 클릭 → 마이크로 따라 읽기
- AI가 발음 정확도를 퍼센트로 표시
- 10번 완벽하게 읽으면 "완성!" 배지 획득
- 숫자, 통화 기호 등 특수 표현도 자동 변환 (€ → "euro")

### 5. 학습 정리 도구

- **요약 생성**: AI가 대화 내용을 어휘/문법/핵심 문장으로 정리
- **노트 저장**: 유용한 AI 답변을 노트로 저장
- **폴더 관리**: 저장한 내용을 폴더/페이지 구조로 정리

### 6. 9개 언어 지원

독일어, 영어(미국/영국), 프랑스어, 스페인어, 이탈리아어, 일본어, 중국어, 포르투갈어

PDF마다 학습 언어를 개별 설정 가능.

## UX 흐름

### 첫 사용 (온보딩)

```
Google 로그인 → PDF 업로드 → 언어 선택 → 바로 질문 시작
```

### 일반 학습 세션

```
1. 앱 접속 → 사이드바에서 PDF 선택 (마지막 페이지에서 자동 이어보기)
2. PDF 보면서 모르는 부분 드래그 → 팝업에서 "질문하기"
3. AI가 교재 맥락 기반으로 답변 (실시간 스트리밍)
4. 답변이 좋으면 👍 / 노트로 저장
5. 세션 끝날 때 "요약 생성" → 복습 자료 확보
```

### 발음 연습 세션

```
1. PDF에서 문장 드래그 → "연습" 클릭
2. 원어민 발음 먼저 듣기 → "연습 시작"
3. 마이크로 따라 읽기 → 정확도 확인
4. 10회 성공 시 완성 배지
```

## 기술적 차별점

| 항목 | LinguaRAG | 일반 AI 챗봇 (ChatGPT 등) |
|------|-----------|--------------------------|
| 교재 기반 답변 | 업로드한 PDF 내용만으로 답변 (RAG) | 일반 지식으로 답변, 교재 맥락 없음 |
| 페이지 인식 | "이 페이지" 말하면 현재 페이지 자동 참조 | 수동으로 텍스트 복붙 필요 |
| 학습 연속성 | 대화 기록 + 마지막 페이지 기억 | 매번 새로운 대화 |
| 발음 연습 | 브라우저 내장 TTS + STT로 즉시 연습 | 별도 앱 필요 |
| 멀티 교재 | PDF별 독립 대화, 독립 RAG | 교재 구분 불가 |
