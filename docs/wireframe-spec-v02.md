# LinguaRAG - Wireframe Specification v0.2

**Document Status:** Draft v1.0
**Last Updated:** 2026-02-27
**Scope:** v0.2 신규 화면 및 v0.1 변경 사항만 기술 (v0.1 전체 명세는 [wireframe-spec.md](./wireframe-spec.md) 참조)
**PRD Reference:** [prd.md](./prd.md) — v0.2 항목: "3-패널 레이아웃 (교재 원문 + 소스 하이라이팅 + Q&A). TTS 반복 듣기 플레이어. Supabase 이전."

## v0.2 주요 변경 요약

| 항목 | v0.1 | v0.2 |
|------|------|------|
| 레이아웃 | 2-패널 (사이드바 + 채팅) | 3-패널 (사이드바 + 소스 패널 + 채팅) |
| RAG 소스 표시 | 없음 | 답변 기반 교재 원문 청크 + 하이라이팅 |
| TTS | 볼륨 슬라이더만 | 인라인 플레이어 (재생/일시정지/반복) |
| 입력 제한 | 카운터 없음 | 500자 카운터 + 초과 시 비활성 |

### TTS 인라인 플레이어 (v0.2 신규)

v0.1의 볼륨 슬라이더를 TTS 인라인 플레이어로 확장. 각 assistant 메시지에 재생 버튼 추가.

#### 컴포넌트 구조

```
┌─────────────────────────────────┐
│ 🔊  0:03 ──●──────────── 0:12  │
│ [◀◀] [▶/⏸] [반복 2×]  Vol: ━● │
└─────────────────────────────────┘
```

#### Key Components (TTS 플레이어)

| 컴포넌트 | 유형 | 목적 | 동작 |
|---------|------|------|------|
| 재생/일시정지 버튼 | Toggle Button | TTS 재생/중단 | 재생 중 아이콘 ▶ → ⏸ 전환 |
| 되감기 버튼 | Icon Button | 현재 문장 처음부터 재시작 | 클릭 시 현재 재생 위치 0초로 리셋 |
| 프로그레스 바 | Seekbar | 재생 위치 표시 및 조정 | 드래그로 재생 위치 조정 |
| 시간 표시 | Static Text | "현재 시간 / 총 시간" | 실시간 업데이트 |
| 반복 버튼 | Toggle Button | 반복 횟수 순환 (1회 → 2회 → 3회 → 무한) | 클릭마다 순환: 1× → 2× → 3× → ∞ |
| 볼륨 슬라이더 | Range Input | TTS 볼륨 조정 | 전역 볼륨. 위치: 우하단 InputBar (v0.1 위치 유지) |
| 메시지 재생 버튼 | Micro Button | 특정 메시지 TTS 재생 | 메시지 우상단 호버 시 표시. 클릭 시 해당 메시지 TTS |

#### 플레이어 위치 및 표시 조건

- **표시 조건**: 마지막 assistant 메시지 아래 자동 표시 (streaming 완료 직후)
- **위치**: 채팅 메시지 목록 내, assistant 메시지 버블 하단
- **이전 메시지**: 각 assistant 메시지 우상단 호버 시 소형 🔊 버튼만 표시

## 수정된 컴포넌트 (v0.1 → v0.2)

### 메시지 입력창 (InputBar) — 500자 카운터 추가

```
┌──────────────────────────────────────────────────────┐
│ 질문을 입력하세요... (Shift+Enter 줄바꿈)         48/500 │
└──────────────────────────────────────────────────────┘
```

| 상태 | 카운터 스타일 | 전송 버튼 |
|------|------------|---------|
| 0~399자 | 회색 (`text-gray-400`) | 활성 |
| 400~499자 | 주황 (`text-amber-500`) | 활성 |
| 500자 | 빨강 (`text-red-500`) | 비활성 |

### 채팅 메시지 — 소스 링크 추가

assistant 메시지 하단에 인용된 교재 소스 링크 표시:

```
┌────────────────────────────────────────┐
│ **Wann passt es dir?**                 │
│                                        │
│ 예문: ...                              │
│ 번역: ...                              │
│ 문법: ...                              │
│                                        │
│ 📖 출처: A1-13 약속하기 · p.47        │  ← 클릭 시 소스 패널 해당 청크 하이라이트
└────────────────────────────────────────┘
```

- 소스가 없는 답변 (컨텍스트 기반 지식 활용 시): 출처 링크 미표시
- 소스가 있는 답변: "📖 출처: {unit_id} {unit_title} · p.{page}" 표시
- 클릭 시: 소스 패널이 열려 있으면 해당 청크로 스크롤 + 1.5초 pulse 애니메이션. 닫혀 있으면 자동 열림 후 스크롤.

## States (v0.2 신규 상태)

### 소스 패널 상태

| 상태 | 설명 |
|------|------|
| **대기 (Idle)** | 이전 답변의 소스 카드 표시. 새 질문 전송 시 유지 |
| **검색 중 (Loading)** | 질문 전송 후 RAG 검색 중. 소스 카드 영역에 스켈레톤 UI (2개 카드) |
| **소스 있음 (Has Sources)** | N개 소스 카드 표시. 하이라이팅 활성 |
| **소스 없음 (No Sources)** | "교재 원문을 찾지 못했습니다" 안내 텍스트. 단원 원문 탭으로 자동 포커스 제안 |
| **접힘 (Collapsed)** | 패널 너비 0. 채팅이 2-패널 너비로 확장. 헤더 좌측에 "교재 원문 ▶" 버튼으로 재열기 |

### TTS 플레이어 상태

| 상태 | 설명 |
|------|------|
| **대기** | 재생 버튼만 표시. 프로그레스 바 0 |
| **로딩** | TTS API 요청 중. 버퍼링 스피너 표시 |
| **재생 중** | 프로그레스 실시간 업데이트. ⏸ 아이콘 |
| **일시정지** | 현재 위치 유지. ▶ 아이콘 |
| **완료** | 프로그레스 바 100%. 반복 설정에 따라 재시작 or 대기 상태로 복귀 |
| **오류** | TTS 실패 시 플레이어 숨김 + 토스트 에러 메시지 |

## Actions & Transitions (v0.2 신규)

| 사용자 행동 | 시스템 응답 | UI 변화 |
|-----------|-----------|--------|
| 질문 전송 | RAG 검색 + Claude 스트리밍 동시 시작 | 소스 패널: 스켈레톤 UI. 채팅: 스트리밍 |
| RAG 결과 수신 | 소스 카드 렌더링 | 소스 패널: 카드 표시, 하이라이팅 활성화 |
| 채팅 "출처 링크" 클릭 | 소스 패널 해당 청크 포커스 | 소스 패널: 스크롤 + pulse. 닫혀 있으면 열림 |
| 소스 카드 "🔊 듣기" 클릭 | 해당 청크 TTS 재생 | 소스 패널 내 미니 플레이어 인라인 표시 |
| assistant 메시지 완료 | TTS 플레이어 표시 | 메시지 하단에 TTS 플레이어 마운트 |
| 반복 버튼 클릭 | 반복 모드 순환 | 버튼 텍스트: 1× → 2× → 3× → ∞ |
| 소스 패널 접기 | 패널 너비 → 0 | 채팅 영역 확장. 헤더에 "교재 원문 ▶" 표시 |
| "단원 원문 전체" 탭 | 현재 단원 전체 텍스트 로드 | 스크롤 가능한 전체 원문. 소스 청크 인라인 하이라이팅 |

## Responsive Behavior (v0.2)

| Breakpoint | 레이아웃 |
|-----------|---------|
| Mobile (<768px) | 단원 트리: 드로어. 소스 패널: 하단 Sheet (드래그로 올림). 채팅: 전체 너비 |
| Tablet (768-1024px) | 소스 패널 기본 접힘. 채팅 헤더 "교재 원문 ▶" 버튼으로 수동 열기 |
| Desktop (>1024px) | 3-패널 기본 레이아웃. 모든 패널 동시 표시 |
